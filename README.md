# ✨ VibeSearch

**Search & Recommend Based on Aesthetic / Intent**

VibeSearch helps you discover places and content that match a **vibe**, not just keywords.

Describe your mood → get **ranked recommendations** like coffee shops, workspaces, or content that fit your aesthetic.

---

## 🚀 Demo

👉 [https://your-vibesearch.vercel.app](https://vibesearch-rust.vercel.app/)

---

##  How It Works

1. User enters a **vibe description**
2. The system processes the input
3. Items are ranked based on **semantic similarity**
4. Results are displayed with:
   - **#1 Top Recommendation**
   - Ranked alternatives

---

### Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS
- TypeScript
- @xenova/transformers (local AI, no API keys)
